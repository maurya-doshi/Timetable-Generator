# 🧱 GENERAL CONSTRAINTS

1. 4 Labs available across all sections
2. Maths and First Year subjects are to be taken as input
3. Teachers must have atleast one free slot b/w occupied slots (can be considered as a soft constraint)
4. Course Code format ~ L:T:P --> Lecture : Tutorial : Practical
   for eg. : 3:0:1 , 1:0:0, etc;
5. L:T:P --> Lecture : Tutorial : Practical
   Lecture -> Duration : 55mins | 2 units of work
   Tutorial -> Duration : 55x2 mins | 2 units of work
   Practical -> 55x2 mins | 2 units of work
   In general, subject with T unit is to be taken in class, but for some subjects the T unit is taken in Lab, but it does not qualify as a P unit, basically Tutorial being taken in Lab.
6. College working hours ~ 9 AM - 4:30 PM
   Short Break ~ 10:50 - 11:05
   Lunch Break ~ 12:50 - 1:45
7. In total 5 working days ~ Monday to Friday
8. Friday is to be a half day compulsory
9. One Lecture slot = 55mins

# 🎓 POSTGRADUATE (PG) CONSTRAINTS

19. PG DETAILS
    Two specializations let's call them : SP-1 and SP-2
    each one having 1 class allotted to them

20. PG Professional Electives (PEs)
    All 3 Professional Elective courses must be scheduled at the exact same time slot for SP-1 and SP-2. The faculty and rooms are different, but the timing is perfectly synchronized.

21. PG Shared Core Course
    One core course is exactly the same across both specializations. It must be conducted by the same faculty, in the same room, at the same time.

22. PG Labs & Tutorials
    All Labs and Tutorials must be scheduled at the exact same time slot for SP-1 and SP-2. The faculty and rooms are different, but the timing is synchronized.
